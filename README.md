# Atlas Private Wealth & Legacy Office

Un sitio web elegante creado con Astro y React para Atlas Private Wealth.

## Tecnologías

- Astro 4.x
- React 18
- Tailwind CSS 3.x
- TypeScript

## Colores del proyecto

- Cream: #f7f3ed
- Dark Blue: #0d1926
- Light Cream: #ebe7e0
- Gold: #d4af37

## Fuente

- CameraPlainVariable

## Desarrollo

```bash
npm install
npm run dev
```

El sitio estará disponible en http://localhost:4321

## Build

```bash
npm run build
npm run preview
```
